## Model Details

* **Architecture**: Deep CNN with 4 conv layers + BatchNorm + Dropout
* **Optimizer**: AdamW
* **Scheduler**: ReduceLROnPlateau
* **Data Augmentation**:

  * Random Horizontal Flip
  * Random Rotation
  * RandomResizedCrop
  * ColorJitter
  * **Mixup** and **CutMix**
